# Schema Form

Based on [React JSON Schema Form](https://github.com/rjsf-team/react-jsonschema-form)

Here's a simple example using the following JSON schema:

```json
{
  "type": "object",
  "properties": {
    "id": {
      "type": "string"
    },
    "date": {
      "type": "string",
      "format": "date"
    },
    "url": {
      "type": "string"
    }
  }
}
```
